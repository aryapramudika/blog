Shell Script Untuk Pemula (Bagian pertama)#

Mengapa perlu belajar shell scripting?#

Seorang system administrator perlu belajar mengenai shell scripting karena akan mempermudah dalam melakukan administrasi suatu system, sebagai contoh sebagai berikut:

  • Melakukan daily backup system
  • Membuat banyak user di system
  • Melakukan instalasi banyak software secara bersamaan
  • Melakukan monitoring / checking suatu layanan

Dengan menggunakan shell script, task-task diatas akan sangat mudah dilakukan.

Di artikel ini saya akan membahas shell scripting secara spesifik menggunakan bash, karena apa? karena bash banyak digunakan

Hal-hal yang perlu disiapkan:

  • Linux (Distro apapun) bisa di install di Laptop/PC/Virtualiasi/Cloud bebas hehe
  • Camilan (maybe)
  • Kopi hitam pahit (buat sajen)

Buat direktori untuk script yang kita buat:

mkdir belajar-shell-scripting
cd belajar-shell-scripting

Membuat shell script sederhana#

Buat dengan nama file halo.sh

vim halo.sh

echo "Halo Dunia"

Untuk menjalankan script yang kita buat, kita perlu menambahkan permission execute(x).

chmod a+x halo.sh

Jalankan script

./halo.sh
Halo Dunia

Variabel#

Variabel berfungsi untuk menyimpan sebuah data yang bisa kita gunakan di dalam program/script.

Variabel dapat diakses dengan menambahkan $ pada nama variabel contoh: $nama

Buat dengan nama file variabel.sh

vim variabel.sh

nama=Arya
echo "Halo $nama"
echo "Bagaimana kabarmu $nama?"

Jalankan seperti tadi

chmod a+x halo.sh

./variabel.sh
Halo Arya
Bagaimana kabarmu Arya ?

!! variabel di Linux bersifat huruf besar/kecil bersifat sensitif, NAMA dengan nama akan di identifikasi sebagai variabel yang berbeda. Jangan sampai salah ya..

Argument#

Argument merupakan opsi/nilai yang bisa kita masukkan ketika menjalankan script/program.

Contoh, kita buat dari script variabel.sh sebelumnya

cp variabel.sh argument.sh

Ubah script

vim argument.sh

Misalkan kita mau melakukan input nama di variabel $nama sesuai dengan keinginan, dan juga kita tidak mau mengubahnya secara manual di script, maka kita bisa menggunakan argument, sebagai berikut:

nama=$1
echo "Halo $nama"
echo "Bagaimana kabarmu $nama ?"

Kita cukup ubah variabel nama dengan nilai $1

Apa sih nilai $1 itu?

$1 merupakan input pertama ketika kita menjalankan program/script, contoh:

./halo.sh arya dika

arya merupakan argument pertama ditandai dengan $1

dika merupakan argument kedua ditandai dengan $2

Selanjutnya jika ada argument ke-3,4,5 maka akan ditandai dengan $3 $4 dan $5

Maka dengan mengubah nilai variabel nama menjadi $1, nanti kita bisa menginput nama sesuai dengan keinginan kita.

Yuk jalankan scriptnya:

./argument.sh Paijo

Halo Paijo
Bagaimana kabarmu Paijo ?

Jalankan lagi

./argument.sh Selamet

Halo Selamet
Bagaimana kabarmu Selamet ?

Read Input#

Bagaimana jika program yang kita buat ingin membaca input dari user dengan ada dialognya, contohnya:

./halo.sh 
Masukkan Nama: Wukir    

Halo Wukir
Bagaimana kabarmu Wukir?

Ayo kita buat

Copy file dari argument.sh

cp argument.sh read-input.sh

Ubah

vim read-input.sh

read -p "Masukkan Nama: " nama

echo "Halo $nama"
echo "Bagaimana kabarmu $nama?"

Coba jalankan:

./read-input.sh 
Masukkan Nama: Moden

Halo Moden
Bagaimana kabarmu Moden?

Fungsi read -p yaitu akan menampilan prompt sebelum membaca input.

Aritmatika (Penjumlahan/Pengurangan/Perkalian/Pembagian)#

Langsung saja kita buat script untuk menjumlahkan 2 angka, seperti berikut:

vim penjumlahan.sh

echo $(( 10 + 20 ))

Jalankan

./penjumlahan.sh
30

Kita juga bisa mengkombinasikan dengan variabel/arguments.

Contoh dengan variabel

X=10
Y=10

echo $(( $X + $Y ))

Jalankan lagi

./penjumlahan.sh
20

Untuk operasi lain sama juga, pengurangan dengan simbol -, perkalian dengan simbol *, pembagian sengan simbol /

Buat script seperti berikut:

vim aretmatika.sh


X=$1
Y=$2

penjumlahan=$(( $X + $Y ))
echo "Hasil penjumlahan dari $X dengan $Y adalah $penjumlahan"


pengurangan=$(( $X - $Y ))
echo "Hasil pengurangan dari $X dengan $Y adalah $pengurangan"


perkalian=$(( $X * $Y ))
echo "Hasil perkalian dari $X dengan $Y adalah $perkalian"

pembagian=$(( $X / $Y ))
echo "Hasil dari pembagian dari $X dengan $Y adalah $pembagian"

Script ini adalah kombinasi materi yang sudah kita pelajari mulai dari variabel, argument, dan operasi aritmatika

Jalankan yokk

./aretmatika.sh 20 10
Hasil penjumlahan dari 20 dengan 10 adalah 30
Hasil pengurangan dari 20 dengan 10 adalah 10
Hasil perkalian dari 20 dengan 10 adalah 200
Hasil dari pembagian dari 20 dengan 10 adalah 2
